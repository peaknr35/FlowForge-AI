# Changelog

All notable changes to FlowForge AI are documented here.

The format follows [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [0.1.0] - 2026-04-20

### Added

- Workflow Builder module: n8n-style automation blueprint generation.
- Business Plan Generator module: structured business plans with financial projections.
- Cold Email Sequencer module: multi-step outreach sequence generation.
- Contract Analyzer module: risk flag identification, clause extraction, and summarization.
- Niche Copy Engine module: audience-targeted marketing copy generation.
- 12 specialized AI agents for research, drafting, review, and quality assurance.
- Ollama local inference integration -- all AI processing runs on the user's hardware.
- Supabase authentication via Google OAuth.
- Stripe subscription billing (Starter, Pro, Agency, Lifetime plans).
- Plan-level generation limits and enforcement.
- Next.js 16 App Router with TypeScript strict mode.
- Tailwind CSS 4 design system.
- Zustand state management and TanStack Query data fetching.
- Zod input validation across API routes.
